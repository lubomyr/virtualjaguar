1) For build you need linux-console

2) Install Android sdk&ndk (i unpacked it into ~/sdk directory)
http://developer.android.com/ndk/downloads/index.html
https://developer.android.com/sdk/index.html

for run android environment i create and use simple script setenv-android.sh

#!/bin/sh

export PATH=~/sdk/android-ndk-r11:$PATH
export PATH=~/sdk/android-sdk-linux/tools:$PATH
export PATH=~/sdk/android-sdk-linux/build-tools/24.0.2:$PATH

now we can run android build environment via console
. setenv-android.sh

Now via Android SDK Manager (for call sdk manager via console type android). You need download latest version:
Android SDK Tools
Android SDK Platform-tools
Android SDL Build tools
Android 6.0 (API 23)
+SDK Platform (API 23)

3) Install pelya's libSDL-environment
cd ~
git clone git://github.com/pelya/commandergenius androidsdl

4) download virtualjaguar sources
cd ~/androidsdl/project/jni/application
git clone git://github.com/lubomyr/virtualjaguar

// Now we can build project (apk)
cd ~/androidsdl
./build.sh virtualjaguar
